# Changelog

All notable changes to the Sealevel VS Code extension.

## [0.1.0] — 2026-04-15

### Added
- `@slm` chat participant for the VS Code Chat panel — ask Solana/Anchor questions inline
- Inline autocomplete for `.rs`, `.ts`, `.tsx`, `.toml` files (debounced, configurable)
- `Sealevel: Explain Error Code` command — paste a Solana error code, get an explanation
- `Sealevel: Explain Transaction` command — paste a signature, get a breakdown
- Settings: `slm.apiKey`, `slm.apiUrl`, `slm.mode`, `slm.autocomplete.enabled`, `slm.autocomplete.debounceMs`
